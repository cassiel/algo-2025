1. Start server: =bb watch-app=: plants port number in =.nrepl-port=

1. Fire up Node.js process in Max (and check =node.script= console) - =figwheel-main.edn= disables any attempt to launch Node
1. =setup.cljs= for event handler and general setup
1. Teardown (=ctrl/unload-all=) prior to patcher save
